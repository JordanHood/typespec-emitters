# TypeSpec Emitters

Monorepo for TypeSpec emitters.

## Packages

- [`@typespec-dev/emitter-fastify`](./packages/fastify) - Fastify server code generation

## Requirements

- Node.js >=22.0.0
- pnpm >=10.0.0

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test

# Run linting
pnpm lint
```
